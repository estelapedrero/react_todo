# React to-do list
To-do list to add tasks sorted by categories, check done/undone and remove.

# Installing the project

### Clone the repository
```
$ git clone https://github.com/estelapedrero/react_todo.git
```

### Access the project
```
$ cd react_todo
```

### Install the dependencies
```
$ npm install
```

### Run the application
```
$ npm start
```

### Lint test
```
$ npm run lint:js
```
